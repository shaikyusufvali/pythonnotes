## Sample Program

```python
import numpy as np

a = np.array([10, 20, 30, 40, 50, 60])

print("Sum:", np.sum(a))
print("Mean:", np.mean(a))
print("Maximum:", np.max(a))
print("Minimum:", np.min(a))
print("Variance:", np.var(a))
print("Standard Deviation:", np.std(a))
print("Square:", np.square(a))
print("Square Root:", np.sqrt(a))
```
